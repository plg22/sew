Caracteristicas principales de Node.js:

Concurrencia

Node.js funciona con un modelo de evaluación de un único hilo de ejecución, sin incurrir en cambios de contexto.

V8

V8 es el entorno de ejecución para JavaScript creado para Google Chrome. Escrito en c++, compila el código fuente javascript

Módulos

Node.js incorpora varios "módulos básicos" compilados en el propio binario.

Desarrollo homogéneo entre cliente y servidor

Node.js puede ser combinado con una base de datos documental o relacional.

Bucle de eventos

Node.js se registra con el sistema operativo y cada vez que un cliente establece una conexión se ejecuta un callback.
